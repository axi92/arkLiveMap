## Prerequisites

- npm >=10.8.3
- node >=20.14.0

## Install

```sh
npm install
```

### Add server

```js
node addserver.js
// Restart service to take effect!
```


## Usage

```sh
npm run start
```

Or with docker:
```sh
docker compose up
```
